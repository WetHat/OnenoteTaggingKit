## Remarks

This function attempts to keep the change notifications to a minimum. **Note**: The [`Tags`](./Tags.md) property is updated with the tags extracted from the given pages. However, the [`Pages`](./Pages.md) property needs to be updated by the caller with a subset of the returned page collection as determined by implementation specific rules.
