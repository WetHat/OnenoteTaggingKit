# TagFilterBase.AutoUodateEnabled property

Get or set the auto-update flag.

```csharp
public bool AutoUodateEnabled { get; set; }
```

## Remarks

Enables or disables the automatic calculateion of refinement filter effects after changes in the filter configuration. Filter effects are represented by the [`FilteredPageCountDelta`](../RefinementTagBase/FilteredPageCountDelta.md) and [`RefinementTagBase`](../RefinementTagBase.md) properties of the refinement tags in this filter.
