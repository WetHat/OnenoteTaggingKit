## Remarks

The panel requires the highlightable controls it hosts to be backed by a data context implementing the <a href="ea720471-b128-4927-e7a0-f4b1418c5ca4">IHighlightableTagDataContext</a> contract. The details of the highlighting are left the specific control implementation. Typically the data context provides a property like 
```
IEnumerable<TextFragment> HighlightedName { get; }
```
 This property can be used by the UI implementation to build text with highlights like so: 
```
private void createHitHighlightedTag(HitHighlightedTagButtonModel mdl)
{
  hithighlightedTag.Inlines.Clear();
  foreach (TextFragment f in mdl.HitHighlightedTagName)
  {
      Run r = new Run(f.Text);
      if (f.IsMatch)
      {
          r.Background = Brushes.Yellow;
      }
      hithighlightedTag.Inlines.Add(r);
  }
}
```


The highlightable control should be defined as a tag template ( <a href="7fedd4ce-5518-004b-3cfa-a8865fc81ef5">TagTemplate</a>). 
**DataTemplate Example**<br />
``` XML
<cui:HighlightedTagsPanel TagSource="{Binding TagSuggestions,Mode=OneWay}"
                    Header="Test">
<cui:HighlightedTagsPanel.TagTemplate>
  <DataTemplate>
    <local:HitHighlightedTagButton Click="OnSuggestedTagClick"/>
  </DataTemplate>
</cui:HighlightedTagsPanel.TagTemplate>
</cui:HighlightedTagsPanel>
```
 the panel instantiates highlightable controls from the data template and assigns a data context from a <a href="ea720471-b128-4927-e7a0-f4b1418c5ca4">IHighlightableTagDataContext</a> implementation .

Implementations of the <a href="66415d03-ea1e-bdf0-d2f6-bd3f122359ba">ITagSource</a> contract are usually based on observable collections of some sort. Such a collection can be directly bound to the <a href="69bfe48a-404c-3d5c-aa5f-fa6a9d28acf1">TagSource</a> property without additional modification, like so: 
**XML**<br />
``` XML
<cui:HighlightedTagsPanel ...
                          TagSource="{Binding TagSourceObservableCollection,Mode=OneWay}"
                          Header="{Binding ...}">
```


A simple implementation of <a href="66415d03-ea1e-bdf0-d2f6-bd3f122359ba">ITagSource</a> looks like this 
**C#**<br />
``` C#
public class TagSource : ObservableSortedList<TagModelKey, string, TagSelectorModel>, ITagSource
{
   #region ITagSource
   public IEnumerable<IHighlightableTagDataContext> TagDataContextCollection
   {
     get { return Values; }
   }
   #endregion
}
```
