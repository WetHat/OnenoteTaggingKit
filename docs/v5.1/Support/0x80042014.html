Exception from HRESULT: 0x80042014

A page no longer exists at that location.

This can happen if you attempt to navigate to a page which is no longer at the location when the link was created. In the context of the Tagging Kit this can happen in these scenarios.

  • In the Pages panel of the Find Pages dialog you clicked on a link to a OneNote page which has beed moved or deleted while the search result was displayed.

    ☝️ To fix this refresh the search result.

  • On a Saved Search you clicked on a link to a page in the search result which was deleted or moved since the search was last updated.

    ☝️ To fix this update by the serch result by executing the Update command or the Update action of the Tag Pages dialog.
