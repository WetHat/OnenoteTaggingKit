OneNote Tagging Kit Installation and Upgrade

  1. Verify that you have a supported version of OneNote. See OneNote Version Check on how to do that.

  2. Make sure _OneNote_is not running! Exit _OneNote_if necessary.

  3. Run the installer downloaded from the Releases page.

    Alternatively you can also install or upgrade the OneNote Tagging Kit via the Chocolatey package manager for Windows. On a PowerShell command prompt type:

Install:

C:\> choco install onenote-taggingkit-addin.install

Upgrade:

C:\> choco upgrade onenote-taggingkit-addin.install
