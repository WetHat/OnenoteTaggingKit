Exception from HRESULT: 0x80042030

A background tagging operation failed because it was blocked by a modal OneNote dialog such as a:

  • sign-in dialog
  • OneNote options dialog
  • A OneNote Customize Tags dialog *...

When this error occurs close the modal OneNote dialog and then close the Tagging Kit error dialog to continue background tagging.
